# 线程
*进程*：运行中的程序
*线程*：进程中的进程
```cpp
#include <iostream>
#include <mingw.thread.h>
#include <string>
void printhelloworld(std::string msg){
    std::cout<<msg<<std::endl;
    return;
}

int main(int argv, char *argc[]){
    std::thread thread1(printhelloworld, "hello thread");
    thread1.join(); //线阻塞调用线程，直到 thread1 所代表的线程执行完成。
    //// 主线程等待 thread1 完成
    return 0;
}
```
